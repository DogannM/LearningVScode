Extensions are like "add-ons" that you can download and install to add new functionality to your code editor.

Support for programming languages

Linters

Debugging tools

Themes

Keymaps

Code snippets

Source control tools

Code formatters

And more!

With just a few clicks, you can find the extensions that fit your needs and install them from the Extensions Marketplace, directly within Visual Studio Code.

To get started with a new project, you'll often need to clone a repository from a remote source.

Visual Studio Code makes this process simple and efficient.

There are two different ways of cloning a repository. Let's see them.

◼️ Cloning From the Explorer
First, you should make sure that you don't have a folder open in Visual Studio Code.

Then, open the Explorer and you'll see different options but you should click on "Clone Repository."


You'll have to enter the URL of the repository that you want to clone. You can also sign it to GitHub and authenticate to choose the repository from a list of repositories.

And another option use git terminal with $git clone "url"


◼️ Cloning From the Command Palette
Alternatively, you can go to the Command Palette with Ctrl+Shift+P (Windows) or Cmd+Shift+P (macOS).

Search for Git: Clone and select this option.


You'll see the same field where you can enter the URL of the repository that you want to clone.


If you press Enter, the cloning process will begin and you'll have your cloned repository ready in a few seconds or minutes.

◼️ Basic Structure
To write a table in Markdown:

Start each row with a pipe symbol (|).

Use three or more hyphens to create each column's header.

End each row with a pipe (|).

| Product  | Price |
|----------|-------|
